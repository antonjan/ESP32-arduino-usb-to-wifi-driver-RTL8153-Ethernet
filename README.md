# ESP32-arduino-usb-to-wifi-driver-RTL8153-Ethernet
This repository will have all the code to allow an ESP32-S3 board to connect using its usb port in OTG mode. The fermware allows you to get IP with DHCP and allow static ip using the usb to ethernet cable asemby with the RTL8153 chipset.
This Repository will have all the libraeries and exsample code to connect to an ethernet router.
The libraeries is made for Arduino debvelopment environment.
